# Chat App

## Description
This is a chat app for mobile devices using React Native. The app will
provide users with a chat interface and options to share images and their
location.

## Dependencies:
- React
- React Native
- Expo
- Gifted Chat library
- Google Firestore Database
- Firebase Cloud Storage
